# Road to Italy

A simple React + TypeScript web app to track kilometers for Ada and Diego on their road to Italy.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## Features

- Track kilometers for two people (Ada and Diego)
- Add kilometers using the + button
- Input validation for decimal numbers
- Modern, responsive UI

## Build

To build for production:
```bash
npm run build
```
